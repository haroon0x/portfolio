- Existing Playwright assertions now match the retained heading and accessible
  mobile navigation, and the PR filter assertion checks actual result statuses.
- Selected work now leads with three evidence-backed system case studies and
  keeps the remaining projects in a compact archive.
- The homepage proof strip derives pull-request, repository, and organization
  counts from the existing PR dataset.
- `scripts/fetch-prs.cjs` now discovers public pull requests with GitHub author
