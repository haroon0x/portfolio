# haroon0x

Digital minimalist portfolio showcasing clean design and purposeful development.

## Features

- Minimal, dark-themed design
- Smooth animations and transitions
- Responsive layout
- Text-focused work showcase
- Clean typography and spacing

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```
